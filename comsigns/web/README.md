# COMSIGNS Web UI

Interfaz web React para el sistema COMSIGNS.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`

## Construcción

```bash
npm run build
```

## Características

- Subida de video mediante drag & drop o selección de archivo
- Visualización de resultados de inferencia
- Información detallada de embeddings generados
- Interfaz moderna y responsive
